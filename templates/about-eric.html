About the Data Project


For this project, we retreived a dataset with over 100,000 rows of global detailed terrorist attacks from 1970 to 2017.

We then removed all unnecessary columns and uploaded the cleaned dataset to Pandas.

A Python app was created that helped us deploy the files to Heroku. The result created the page you are viewing now!
